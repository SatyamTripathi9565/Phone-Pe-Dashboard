# 📊 PhonePe End-to-End Data Analysis Dashboard

An interactive Power BI dashboard project built to analyze PhonePe transactions, payment performance, and multiple digital financial services.

---

## 🚀 Project Objective
The goal of this project is to perform end-to-end data analysis on PhonePe transaction data and generate meaningful business insights using Power BI.

## 📌 Dashboard Modules
### 🏠 Home Dashboard

- Service-wise Performance
- KPI Metrics

### 🛡 Insurance Analysis
- Health Insurance
- Car Insurance
- Term Life Insurance
- Monthly Insurance Trends

### 💳 Loans Analysis
- Personal Loans
- Auto Loans
- Loan Transaction Insights

### 💸 Money Transfer Analysis
- Transaction Volume
- Transfer Amount Trends
- Success & Failure Monitoring

### 📱 Recharge & Bills Analysis
- Recharge Categories
- Bill Payment Insights
- Monthly Payment Trends
